WebRTC

WebRTC stands for Web Real-Time Communication and is a great way for communicating with others in a 100% secured environment, without installing external plugins.

To use this application, you need to

  • allow your browser to use your webcam
  • provide your name and click "Ok"

Then you can call other users and chat with them via video and text.
